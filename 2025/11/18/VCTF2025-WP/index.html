利用 Jinja 的 attr 过滤器 + 字符串拼接绕过，Jinja 有内置过滤器 attr，作用类似 Python 的 getattr：

逐步解释：

  1. config.from_envvar：拿到 Config.from_envvar 这个方法
  2. | attr('__glo' ~ 'bals__')：
    • '__glo' ~ 'bals__' 在模板执行时变成 '__globals__'
    • attr('__globals__') 就是 getattr(config.from_envvar, '__globals__')
    • 得到的是 function 的 __globals__ 字典

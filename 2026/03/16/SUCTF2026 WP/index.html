**利用 root 身份运行的 Caddy 直接读 /root/flag**。
日志里最关键的是：localhost:2019 的 Caddy admin API 开着，而且 enforce_origin:false，当前有 grafana shell，就能直接从本机去改 Caddy 配置

思路：

  • Caddy 是 root 跑的
  • Caddy admin API 在 127.0.0.1:2019 开着
  • 日志显示：
    • enforce_origin:false
    • 周期性有 curl/7.81.0 对 /load 发 POST
  • 所以可以从本机直接改 Caddy 配置，让 root 身份的 Caddy 去读 /root/flag
